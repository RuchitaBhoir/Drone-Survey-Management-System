#** Drone-Survey-Management-System**
- A full stack web development application for managing drone survey missions
- It provides a complete solution for planning, monitoring, and managing drone
operations with real-time visualization capabilities.

#**Purpose**
• Plan and create drone survey missions
• Monitor drone fleet status and battery levels
• Track mission progress in real-time
• Visualize flight paths on interactive maps
• Manage mission lifecycle (plan, start, pause, resume, abort)

#**Key Highlights**
• Full-stack application with clean architecture
• RESTful API design
• Real-time mission monitoring
• Interactive map visualization with Leaflet
• Component-based React frontend
• Layered backend architecture
• MySQL database with proper schema design
